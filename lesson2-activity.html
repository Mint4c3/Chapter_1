Creating Bootable USB using Rufus

Instruction:Giving the followings tools and equipment, you should be able to create bootable USB using PowerISO given step-by-step procedure.

Equipments:

  • • USB stick
  • • ISO Image of Windows Server 2008
  • • Desktop Computer / Laptop Computer

Steps / Procedure: (RUFUS)

  • 1. Connect the USB.
  • 2. Open the RUFUS program.
  • 3. Select the CD Icon highlited as shown below, which will let you browse for the required ISO File.

4. Select the file by left clicking on it once and then clicking on Open.

5. Click OK, to start

6. Wait for the program to finish

  • • Show how to download a Windows 10 ISO file from the official Microsoft website (or an alternative legal source).

  • • Show the download process and where the file is saved.

4. Preparing the USB Drive (2:00 - 2:30)

  • • Explain the importance of backing up any data on the USB drive, as the process will erase all data.
  • • Show how to insert the USB drive into the computer

  • • Open the Rufus application.
  • • Explain the main interface of Rufus.

    • - Choose the partition scheme (GPT or MBR) and target system. (Briefly explain the difference if possible, or mention that the default is usually fine for most users).

What to Consider?

In this lesson, students learn how to prepare a bootable installer, a key skill in Computer System Servicing. A bootable installer is typically used to install or repair operating systems on computers, especially when the internal system is damaged or unavailable.

  1. 1. Understanding Bootable Installers:
    • • A bootable installer is a device (usually a USB or DVD) that holds the installation files for an operating system.
    • • It allows the user to boot up a computer and install or repair the operating system, bypassing the need for a working OS on the device itself.
  2. 2. Creating a Bootable Installer:
    • • Tools Needed: Software like Rufus or the Windows USB/DVD Tool helps in creating a bootable USB drive.
    • • Process: The lesson covers how to prepare a USB drive by formatting it and then transferring the OS image onto it.
    • • The learner will also learn how to set up the USB drive to be bootable, ensuring it has the necessary boot sector.
  3. 3. Using Bootable Installers for OS Installation:
    • • Once prepared, the bootable installer is used to install the operating system on a new or repaired computer.
    • • The lesson emphasizes the importance of correctly configuring the BIOS or UEFI settings to allow the system to boot from the USB drive.
  4. 4. Customization and Advanced Options:
    • • In addition to the basic OS installation, students explore how to include extra drivers or software in the installation process, making the installation experience more efficient.
  5. 5. Practical Application:
    • • Students will gain hands-on experience by preparing bootable installers and performing OS installations, learning to troubleshoot and resolve system issues.
