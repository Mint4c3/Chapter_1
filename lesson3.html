LESSON 3

WHAT IS IT


What is Operating System?

Operating System (OS) is the software that manages computer hardware and software resources, providing common services for applications. Installing an OS involves setting up the system’s boot process, preparing storage devices, and ensuring all peripherals function properly by installing drivers.

What is Boot System?

Boot System is the process that starts a computer and loads the operating system into memory. The boot process initializes hardware components and prepares the system for user interaction. How Boot System works?

  • • The computer is powered on, and the BIOS/UEFI firmware initializes.
  • • The firmware performs a Power-On Self-Test (POST) to check hardware integrity
  • • The bootloader searches for the operating system in storage devices.
  • • The OS is loaded into memory, and system processes begin.

Importance of Boot System

  • • Ensures hardware components are functioning properly before the OS loads.
  • • Allows users to configure boot priorities through BIOS/UEFI settings.

Partition and Formatting

Partitioning is the process of dividing a storage device into separate sections, called partitions, to manage data efficiently. Formating prepares a partition by creating a file system that allows the OS to store data.

Importance of Partition and Formatting

  1. • Ensures proper storage allocation and organization.
  2. • Required for installing an OS on a new or unformatted disk.

Common File Systems:

  1. 1. NFTS (Windows) – Supports large files and security features.
  2. 2. EXT4 (Linux) – Optimized for Linux systems, provides fast access.
  3. 3. FAT32 – Compatible with multiple operating systems but has file size limitations.

Step-by-step:

  1. 1. Open the OS installation setup and choose "Custom Installation."
  2. 2. Select the storage drive where the OS will be installed.
  3. 3. Create a new partition by specifying the size and selecting the file system (NTFS for Windows, EXT4 for Linux)
  4. 4. Format the newly created partition to prepare it for installation.
  5. 5. Confirm the partition settings and proceed with OS installation.

Installation of Opearting System

Windows Installation

  1. 1. Insert a bootable USB/DVD and start the computer.
  2. 2. Configure BIOS/UEFI to boot from the installation media.
  3. 3. Select language, time, and keyboard settings.
  4. 4. Choose "Install Now" and enter the product key (if applicable).
  5. 5. Select "Custom Installation," partition the drive, and format it.
  6. 6. Follow the prompts and wait for installation completion.
  7. 7. Configure user accounts, network settings, and privacy preferences.

Linux Installation

  1. 1. Boot from a Linux USB/DVD installer.
  2. 2. Choose "Try Linux" or "Install Linux."
  3. 3. Select the installation type (Erase disk, Dual Boot, or Manual Partitioning).
  4. 4. Configure partitions and set up a root password.
  5. 5. Follow on-screen instructions and restart after installation.

Post-Installation Task

After installing the operating system, several steps must be completed to ensure the system runs smoothly and all hardware components function properly.

  1. 1. Install Device Drivers - Ensure all necessary drivers for the motherboard, graphics card, sound, network, and peripherals (e.g., printers, scanners) are installed and updated.
  2. 2. Run System Updates – Check for and install any available updates to fix bugs, improve security, and enhance performance.
  3. 3. Configure System Setting – Adjust display resolution, sound settings, power options, and network configurations according to user preferences.
  4. 4. Install Essential Software – Set up commonly used applications such as browsers, office tools, and security software.
  5. 5. Create a System Backup– Set up restore points or backup the system to prevent data loss in case of failure.
