Apply and Perform

Performance Task: Installing and Using WinRAR for File Management

Instructions:

1. Prepare the material

  • ● Download the WinRAR installer from the official website (rarlab.com).
  • ● Gather a set of files (documents, images, etc.) to use for testing.

2. Install

  • ● Run the downloaded installer.
  • ● Follow the installation wizard, choosing appropriate options (installation path, file associations, etc.).
  • ● Complete the installation.

3. Use WinRAR for File Management

  • ● Create a compressed archive (.rar or .zip) of the gathered files.
  • ● Extract the files from the archive to a different location.
  • ● Encrypt the archive with a password.
  • ● Create a self-extracting archive (.exe).

4. Submit a Simple Report (1-2 pages)

  • ● What version of WinRAR did you install?
  • ● Describe the steps you took to install WinRAR.
  • ● How you created a compressed archive, extracted files, encrypted the archive, and created a self-extracting archive.
  • ● Did you encounter any issues? How did you resolve them?
  • ● Attach 1-2 screenshots of the WinRAR interface and your compressed/extracted files.

Rubrics

Please be guided with the rubrics below.

Criteria Complete (2) Partial (1) Not Complete (0)
1. Installation WinRAR installed successfully with correct options. WinRAR installed but with some incorrect options. WinRAR installation failed.
2. Compression Archive created successfully with proper settings. Archive created but with incorrect settings. Archive creation failed.
3. Extraction Files extracted successfully. Files partially extracted or with errors. File extraction failed.
4. Encryption Archive encrypted with a password successfully. Archive encrypted but with issues. Encryption failed or not attempted.
5. Self-Extraction Self-extracting archive created successfully. Self-extracting archive created but with issues. Self-extracting archive creation failed.
6. Report Quality Report is clear, detailed, and includes screenshots. Report is incomplete or lacks detail/screenshots. Report is missing or significantly incomplete.
