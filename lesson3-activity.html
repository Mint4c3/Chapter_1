Apply and Perform

Performance Task: Installing an Operating System and Peripheral Device Drivers

Instructions:

1. Prepare the material

  • • Choose an operating system (e.g., Windows or Linux).
  • • Select one peripheral device (e.g., printer, mouse, keyboard, or webcam).

2. Basic OS Installation Steps

  • • Boot from the installation media (USB or DVD).
  • • Follow the installation wizard (choose language, partition disk, and install).
  • • Complete the installation and restart the computer.

3. Install a Driver for a Peripheral Device

  • • Plug in the device (e.g., printer or mouse).
  • • Check if the device is automatically recognized.
  • • If not, manually install the driver by downloading it from the manufacturer’s website.

4. Submit a Simple report (1-2 pages)

  • • What OS did you install?
  • • What peripheral device did you install the driver for?
  • • Did you face any problems? How did you fix them?
  • • Attach 1-2 pictures or screenshots of your process.

Assessment

Directions: On the space provided before each number, write T if the statement is correct; otherwise, write F.

1. The BIOS/UEFI is responsible for initializing hardware components before the OS loads.
2. Partitioning a hard drive is not necessary before installing an OS.
3. The bootloader is responsible for loading the OS into memory.
4. Formatting a partition means erasing all data and setting up a file system.
5. Drivers are software that allow the OS to communicate with hardware devices.
6. NTFS is a common file system used by Linux OS.
7. A missing or outdated driver can cause peripherals like printers and keyboards to malfunction.
8. The first step in OS installation is booting the system from installation media.
9. An OS cannot be installed on an external hard drive.
10. UEFI is the modern replacement for BIOS with enhanced security features.
